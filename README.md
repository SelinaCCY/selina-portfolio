# Selina's Portfolio

A journey-based personal portfolio site for Selina Cheah.

## Tech Stack

This portfolio is built with **React** + **TypeScript**, bundled with **Vite**.

Designed and developed with the help of **Claude Code** and **Claude Design** by Anthropic.

## Running locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).
